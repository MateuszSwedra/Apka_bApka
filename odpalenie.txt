Wymagania wstępne:
Node.js (v18+)
Docker Desktop
Expo CLI
Aplikacja Expo Go na telefonie (iOS/Android)

1. Uruchomienie bazy danych:
docker-compose up -d

2. Konfiguracja środowiska (Backend):
W folderze "backend" utwórz plik .env:
DB_HOST=localhost
DB_PORT=5432
DB_USER=bapka_user
DB_PASSWORD=bapka_password
DB_NAME=bapka_db

3. Konfiguracja środowiska (Frontend):
W folderze "frontend" utwórz plik .env i podaj adres IP swojego komputera:
EXPO_PUBLIC_API_URL=http://192.168.X.X:3000

4. Uruchomienie backendu:
cd backend
npm install
npm run start:dev

5. Uruchomienie frontendu:
cd frontend
npm install
npx expo start
(Zeskanuj kod QR aplikacją Expo Go, telemetria/API idzie po lokalnym IP podanym w .env)
